Folder Structure Copier - PyQt6 App

This package contains a complete Python GUI tool to replicate folder structures and blank files.

CONTENTS:
- folder_structure_gui.py     - The main application
- build_folder_gui.bat        - One-click script to build an EXE using PyInstaller
- README.txt                  - This file

INSTRUCTIONS:
1. Install dependencies:
   pip install pyqt6 pyinstaller

2. Build the app:
   Run `build_folder_gui.bat`

3. Your EXE will appear in the `dist` folder.
